ejemplos

Principales tecnologías y herramientas para el diseño Frontend.

  • HTML:
    Es un lenguaje descriptivo básico. HTML se utiliza para crear el esqueleto de un sitio web. Su uso esta enfocado a crear encabezados, listas, añadir imágenes y tablas, insertar hipervínculos, etc. se basa en crear etiquetas hechas de símbolos.

  • CSS:
    Con el lenguaje CSS podemos trabajar la parte visual de nuestro sitio web. Lo usamos principalmente para diseñar y describir cómo se deben mostrar los elementos HTML. Gracias a este lenguaje podremos personalizar los colores, fondos, animaciones y más de los elementos además de que nos permite elegir dónde colocamos cada cosa.

  • JavaScript:
    JavaScript es un lenguaje utilizado para agregar funcionalidad adicional a un sitio web. Se usa para mejorar las interacciones y agregar nueva funcionalidad, puede usarse para integrar juegos, películas interactivas y otros tipos de archivos.

  • JQuery:
    Es una librería que reúne extensiones y plugin para facilitar el desarrollo. Te ayuda a ahorrar tiempo y a generar código de calidad de forma más sencilla agregando funciones extras como por ejemplo la posibilidad de añadir una animación de carga entre páginas.

Otros Lenguajes usados en Desarrollo Web:


PHP

Es un lenguaje de programación que se usa principalmente en back-end para hacer un puente entre un HTML común y un JavaScript o incluso una base de datos de MySQL.

Phyton

Se usa el lenguaje Python porque es eficiente y fácil de aprender, además de que se puede ejecutar en muchas plataformas diferentes

Java

Este lenguaje se usa por su facilidad y compatibilidad con todo tipo de dispositivos, cualquiera puede aprender a usar Java, No es recomendable para programas muy pesados a o complejos por que da un rendimiento bastante pobre.

Para que sirve un Framework

Un Framework es un conjunto de reglas que se utilizan para desarrollar software de forma más rápida, cómoda y eficiente. Estos marcos se utilizan para ahorrar tiempo y esfuerzo en el desarrollo de aplicaciones al proporcionar una estructura que puede usarse como punto de partida, además de que agrega varias funcionalidades que son bastante útiles para trabajar con CSS.
